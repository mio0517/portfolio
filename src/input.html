研修で学んだこと

私は以下のプログラミング言語を学びました

Linux

Linuxとは…
コンピュータ上でアプリケーションを動かすために必要なOSの一つ
メリット
オープンソースの為コストがかからない・技術が枯れている為１度習得すると長く使用できる
デメリット
サポートが無い為トラブルが発生した場合には、自分で調べる必要がある

HTML

HTMLとは…
Webページの文字やデザインをブラウザから読めるよう、表示する為に使用されている言語
メリット
文字のフォントや色、大きさを変えられる、リンクや画像を貼れる デメリット
動的な表現を作ることはできない

CSS

CSSとは…
HTMLと組み合わせて使用することで、Webサイトの色やサイズ、レイアウト(見た目)のデザインを定義するための言語
メリット
デザインを一括で変更できる、メディアごとに表示スタイルを変更できる
デメリット
ブラウザによって再現性が異なる

Python

Pythonとは…
ブロックチェーン開発、AI開発に向いている言語
メリット
コードの記述が短くて済む為コードが読みやすい
デメリット
大規模開発に向いていない

Java

出来ること:
Webサービス・Android向けアプリの開発に向いている言語 メリット:
コンパイル型の為実行速度が速くデータ処理に優れている デメリット:
小規模システム開発に適していない

Angular

Angularとは…
オープンソースWebアプリケーションフレームワークの一つ
メリット
Webアプリケーションの開発でとてもよく使用されていて、SPAの開発に向いている
フルスタックフレームワークである為、Angularだけ用意すればWebフロントエンド開発ができる設計になっている デメリット
コンポーネント志向の為ファイルが多くなり、関連付けが難しい
